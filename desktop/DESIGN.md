# SafeSquare 데스크톱 클라이언트 — 디자인 작업 명세

> 이 문서는 데스크톱 클라이언트(Windows `.exe` / macOS `.dmg`)의 **디자인이 필요한 모든 항목**을 정리한 것입니다.
> 각 항목의 현재 상태 · 필요 규격(치수/포맷) · 우선순위를 명시했습니다.
>
> **범위 구분 (중요)**
> - 🖥️ **데스크톱 셸(Shell)** = Electron이 직접 그리는 네이티브 화면/아이콘/설치 화면 → **이 문서의 대상**
> - 🌐 **웹 콘텐츠** = 앱 창 안에 뜨는 SafeSquare 웹앱(로그인/대시보드 등) → 별도 프로젝트(`frontend/src`)에서 디자인. 이 문서 대상 아님.
>
> 즉 "앱을 열었을 때 웹 화면이 뜨기 전·연결 실패·설정·아이콘·설치 과정"이 데스크톱 셸 디자인입니다.

## A. 브랜딩 에셋 (빌드에 필수 — 없으면 기본 Electron 아이콘)

| # | 항목 | 현재 상태 | 필요 규격 | 우선순위 |
|---|------|-----------|-----------|----------|
| A1 | **앱 아이콘 (마스터)** | 기본 Electron 아이콘 | **1024×1024 PNG** (투명배경, 모서리 여백 포함). 이 한 장으로 mac/win 아이콘 자동 생성 가능 | ★★★ 필수 |
| A2 | macOS 아이콘 `.icns` | 없음 | A1에서 생성 (16/32/64/128/256/512/1024). macOS는 **둥근 사각형 마스크가 자동 적용 안 됨** → 아이콘 자체에 macOS 스타일 라운드 처리 권장 | ★★★ |
| A3 | Windows 아이콘 `.ico` | 없음 | A1에서 생성, **16/24/32/48/64/128/256 멀티 사이즈 포함** | ★★★ |
| A4 | 트레이/메뉴바 아이콘 | 없음(트레이 미사용) | 트레이 기능 추가 시: mac **16×16 / 32×32(@2x) 흑백(template) PNG**, win **16×16 ICO**. 단색 실루엣 | ★☆☆ (기능 추가 시) |

## C. 데스크톱 셸 화면 (Electron이 직접 그리는 화면) — **핵심 디자인 대상**

> 현재는 전부 인라인 HTML로 임시 제작돼 있어, 디자인 새로 필요.

| # | 화면 | 현재 상태 | 디자인 필요 내용 | 우선순위 |
|---|------|-----------|------------------|----------|
| C1 | **스플래시 / 로딩 화면** | 없음 (연결될 때까지 빈 화면 = 어두운 배경만) | 앱 실행 직후 서버 연결 동안 보여줄 화면. 로고 + 로딩 인디케이터 + "연결 중…" | ★★★ |
| C2 | **연결 실패 화면** | 임시 다크 HTML (`main.js` showErrorPage) | 서버 접속 불가 시 화면. 아이콘/일러스트 + 메시지 + "다시 시도" 버튼 + "서버 주소 변경" 버튼. 현재 서버주소 표시 | ★★★ |
| C3 | **서버 주소 설정 화면** | 임시 HTML 모달 (460×200) | 접속 주소 입력/변경. 입력필드 + 저장/취소 + (선택) 최근 접속 목록·연결 테스트 버튼 | ★★☆ |
| C4 | **정보(About) 화면** | OS 기본 다이얼로그 | 앱 이름/버전/서버주소/저작권. 네이티브 다이얼로그 유지 or 커스텀 창 | ★☆☆ |
| C5 | **업데이트 알림 UI** | 없음 (자동업데이트 미구현) | 자동 업데이트 도입 시: "새 버전 있음 / 다운로드 중 / 재시작하여 적용" | ★☆☆ (기능 추가 시) |
| C6 | **오프라인/네트워크 끊김 배너** | 없음 | 사용 중 연결이 끊겼을 때 상단 배너 등 (선택) | ★☆☆ |

## F. 화면 상태(State) 체크리스트 — 디자인 시 모두 커버 권장

데스크톱 셸이 가질 수 있는 상태들 (각 상태별 화면 필요 여부 판단):
1. **최초 실행 / 온보딩** (서버 주소 미설정 시) — 필요 여부 결정
2. **로딩 중** (C1)
3. **정상 연결** → 웹 콘텐츠 표시 (셸은 투명, 웹앱 디자인 영역)
4. **연결 실패 / 서버 다운** (C2)
5. **인증/세션 만료** (웹앱 영역이나, 셸 차원 처리 필요 시)
6. **업데이트 가능 / 진행 중** (C5)

## 산출물 전달 형식 (디자이너 → 개발 반영용)

- 아이콘: **1024×1024 PNG 1장**이면 mac/win 아이콘 자동 생성 (별도 .icns/.ico 안 만들어도 됨)
- 화면(C1~C4): **HTML/CSS로 구현** 예정이므로 → 시안(Figma/PNG) + 컬러 HEX·폰트·여백 수치, 또는 직접 HTML/CSS
- 설치 배경(B1): PNG, NSIS 배너(B4/B5): BMP (PNG 주면 변환 가능)
- 모든 이미지 @2x(고해상도) 함께 권장
